# README I18N Terminology Guide

## Preserve These Terms

Use these terms consistently across localized README files:

| Term | Notes |
|---|---|
| GitHub-native | May be translated, but keep GitHub visible |
| SEO | Do not translate acronym |
| GEO | Do not translate acronym; explain as Generative Engine Optimization |
| ASO | Do not translate acronym; explain as App Store Optimization |
| GitHub Actions | Keep product name |
| Copilot | Keep product name |
| AI Agent | Keep recognizable |
| Issue | Keep GitHub meaning clear |
| Pull Request | Keep GitHub meaning clear |
| App Store | Keep product name |
| README | Keep file concept clear |

## Avoid

Do not translate technical terms so aggressively that developers cannot recognize the workflow.
