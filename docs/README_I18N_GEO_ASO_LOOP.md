# I18N GEO / ASO Loop

Localized README files are the first layer of a multilingual GEO / ASO loop.

## Loop

```text
localized README -> localized docs -> localized GEO prompts -> localized ASO drafts -> regional measurement -> next language iteration
```

## Example

For Japanese:

```text
README.ja.md
-> Japanese docs and examples
-> Japanese GEO prompt set
-> Japanese App Store subtitle and screenshot copy drafts
-> Japanese market measurement
```

## Required Controls

- use source README as truth
- avoid unsupported local claims
- preserve safety boundary
- track local language changes
- review translations before promotion

## Future GeoLoop Feature

GeoLoop should eventually suggest localized content tasks when:

- a language README exists
- target market is configured
- localized search demand is found
- product fit is high
- risk is low
