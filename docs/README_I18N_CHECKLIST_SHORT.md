# Short I18N README Checklist

Before merging README localization changes:

- [ ] `README.md` remains source of truth
- [ ] localized README files include language links
- [ ] product definition is consistent
- [ ] SEO / GEO / ASO explanations are consistent
- [ ] current status is accurate
- [ ] safety boundary is present
- [ ] roadmap links work
- [ ] no fake local claims
- [ ] no fake rankings or benchmarks
- [ ] no automatic App Store submission claims
