# README I18N Distribution Note

Localized README files should support distribution, not create noise.

## Suitable Distribution Channels

- GitHub repository page
- GitHub topics and search
- developer communities
- Product Hunt launch materials
- local indie hacker communities
- regional App Store localization planning
- AI search / GEO prompt experiments

## Not Suitable

Do not spam links to localized README files in unrelated communities.

## Best Use

Use localized README files as credible reference pages when the audience actually uses that language.

## Measurement

Future measurement can track:

- referral traffic by locale
- GitHub stars after local posting
- AI answer mentions by language
- App Store metadata ideas by locale
- localized documentation engagement
