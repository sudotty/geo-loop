# README I18N Metrics

## What to Measure

Localized README performance can be measured through:

- GitHub repository referrals
- search impressions by language
- AI answer mentions by language
- stars/forks after local distribution
- README link clicks from external communities
- App Store localization idea reuse

## Early-Stage Proxy Metrics

Before full analytics exist, use proxies:

- localized README files exist
- required sections exist
- links are valid
- local language communities respond
- AI search can identify the project in that language

## Do Not Overfit

Localized README files are one asset in the growth surface. They are not a complete localization strategy by themselves.
