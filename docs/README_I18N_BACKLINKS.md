# README I18N Backlinks Note

Localized README files should link to each other and to core project docs.

## Required Links

Each localized README should link to:

- `README.md`
- `README.zh-CN.md`
- `README.ja.md`
- `README.ko.md`
- `README.es.md`
- `ROADMAP.md`
- `docs/PRODUCT_REQUIREMENTS.md`
- `docs/USER_FLOW.md`
- `docs/ARCHITECTURE.md`

## Why This Matters

Backlinks help:

- users switch language
- crawlers discover localized entry points
- AI systems connect language variants to the same entity
- maintainers detect missing or stale links

## Future Check

The i18n scanner should report missing language backlinks.
