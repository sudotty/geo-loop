# README I18N Language Codes

| Locale | File | Language |
|---|---|---|
| en | `README.md` | English |
| zh-CN | `README.zh-CN.md` | Simplified Chinese |
| ja | `README.ja.md` | Japanese |
| ko | `README.ko.md` | Korean |
| es | `README.es.md` | Spanish |

## Naming Rule

Use root-level files:

```text
README.<locale>.md
```

Exception:

```text
README.md
```

is the English source of truth.
