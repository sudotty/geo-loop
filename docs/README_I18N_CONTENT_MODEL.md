# README I18N Content Model

Every localized README should preserve the same content model.

## Model

```text
Language navigation
Product definition
What is GeoLoop
SEO / GEO / ASO explanation
Why GitHub
Core loop
Target users
Current status
Roadmap links
Safety boundary
License
```

## Stable Claims

Stable claims that may be localized:

- GeoLoop is GitHub-native.
- GeoLoop is for SEO, GEO and ASO growth automation.
- GeoLoop turns signals into Issues and PRs.
- GeoLoop uses human review for risky changes.
- GeoLoop does not guarantee AI search ranking.
- GeoLoop does not automatically submit App Store changes by default.

## Unstable Claims

Avoid or review carefully:

- user count
- ranking
- market share
- benchmark numbers
- customer logos
- revenue
- App Store results
- AI engine visibility results
