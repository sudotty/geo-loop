# README Language Navigation Snippet

Add this near the top of `README.md` when updating the main README.

```md
## Languages

- [English](README.md)
- [简体中文](README.zh-CN.md)
- [日本語](README.ja.md)
- [한국어](README.ko.md)
- [Español](README.es.md)
```

This should be placed after the opening product definition and before the main product explanation.
