## Product Definition Pattern

```text
GeoLoop is a GitHub-native SEO / GEO / ASO growth automation loop.
```

Localized versions should preserve:

- GitHub-native
- SEO
- GEO
- ASO
- growth automation loop

## Safety Pattern

```text
Attack the information gap, not the trust system.
```

Localized versions may translate the sentence, but must preserve the meaning.

## Current Status Pattern

```text
This project is currently an executable scaffold, not a complete production crawler.
```

Do not overstate the current implementation stage in any language.
