# README I18N Source of Truth

## Canonical Source

`README.md` is the canonical source of truth.

## Localized Derivatives

The following files are localized derivatives:

- `README.zh-CN.md`
- `README.ja.md`
- `README.ko.md`
- `README.es.md`

## Update Rule

When `README.md` changes materially, localized files should be updated in the same PR or tracked in a follow-up issue.

## Material Changes

Material changes include:

- product definition
- CLI commands
- workflow behavior
- roadmap phases
- safety rules
- hosted product strategy
- supported integrations
- license or legal status

## Non-Material Changes

Non-material changes may not require immediate localization:

- minor typo fixes
- formatting changes
- internal wording polish
- additional examples that are not central to positioning
