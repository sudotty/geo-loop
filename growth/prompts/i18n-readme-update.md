# I18N README Update Prompt

Use this prompt when asking an AI agent to update localized README files.

```text
You are updating GeoLoop localized README files.

Source of truth:
- README.md

Localized files:
- README.zh-CN.md
- README.ja.md
- README.ko.md
- README.es.md

Task:
Update localized README files so they preserve the same product logic, safety boundaries, current status and roadmap links as README.md.

Rules:
- Do not create fake local claims.
- Do not invent customers, rankings, benchmarks, citations or App Store results.
- Keep SEO, GEO and ASO terms recognizable.
- Keep the language natural for developers and founders.
- Preserve the safety boundary.
- Keep links valid.
- If uncertain, mark the wording as needing human review rather than inventing.
```
