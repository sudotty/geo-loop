# I18N README Review Prompt

Use this prompt to review localized README files.

```text
Review the localized README file for GeoLoop.

Check:
1. Does it preserve the same product definition as README.md?
2. Does it explain SEO, GEO and ASO correctly?
3. Does it preserve the GitHub-native growth loop?
4. Does it include target users?
5. Does it describe current status accurately?
6. Does it link to roadmap and docs correctly?
7. Does it preserve safety boundaries?
8. Does it avoid fake local claims, fake customers, fake benchmarks and fake rankings?
9. Is the wording natural for developers and founders in this language?
10. Are there any sections that should be marked for human review?

Return:
- pass/fail
- issues found
- suggested edits
- risk level
```
